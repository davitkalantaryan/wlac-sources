# WLAC stands for Windows to Linux Api Converter

This is library for windows. The library implemets some POSIX functions as well some unix specific
c functions. The implemented functions and corresponding headers (unistd.h, pthread.h, ...)
will allow to compile some sources on WINDOWS, that were developed using pure UNIX and
POSIX functionalities
